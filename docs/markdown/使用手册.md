# 图片管理网站 — 使用手册

---

## 1. 系统简介

图片管理网站是一个功能完善的在线图片管理平台，支持图片上传、浏览、编辑、标签管理和 AI 智能描述生成。系统采用响应式设计，可在 PC 和移动设备上流畅使用。

---

## 2. 快速开始

### 2.1 访问系统

- **前端地址**: http://localhost:5173（开发模式）或 http://localhost:3000（Docker 部署）
- **后端地址**: http://localhost:8000

### 2.2 注册账号

1. 点击页面右上角的「注册」按钮
2. 填写注册信息：
   - **用户名**: 6 位以上字符
   - **邮箱**: 有效的邮箱地址（需唯一）
   - **密码**: 6 位以上字符
   - **确认密码**: 再次输入密码
3. 点击「注册」按钮完成注册
4. 注册成功后自动跳转到登录页面

### 2.3 登录系统

1. 在登录页面输入邮箱和密码
2. 点击「登录」按钮
3. 登录成功后跳转到首页

---

## 3. 功能使用

### 3.1 首页

首页展示系统中的公开图片轮播，以及热门标签云和最新上传的图片。

### 3.2 图片库

点击导航栏的「图片库」按钮进入公开图片库页面，可浏览所有用户公开分享的图片。

#### 3.2.1 浏览公开图片

- 以网格形式展示所有公开图片的缩略图
- 支持滚动加载更多图片
- 显示图片标题和上传者信息
- 点击图片可进入详情页查看完整信息

#### 3.2.2 搜索与筛选

- **关键词搜索**: 搜索图片标题、描述中的关键词
- **标签筛选**: 点击标签可筛选含有该标签的图片
- **热门标签**: 页面展示热门标签云，点击可快速筛选
- **时间筛选**: 选择上传时间范围
- **持有筛选**：选择是否屏蔽自己的图片

#### 3.2.3 排序方式

支持按以下方式排序：
- 上传时间
- 文件大小

#### 3.2.4 查看详情

点击任意图片可查看：
- 完整尺寸的图片预览
- 图片标题和描述
- 上传者信息
- EXIF 拍摄信息
- 相关标签

### 3.3 上传图片

1. 点击导航栏的「上传」按钮进入上传页面
2. 支持两种上传方式：
   - **点击上传**: 点击上传区域选择文件
   - **拖拽上传**: 直接拖拽图片文件到上传区域
3. 支持的格式: JPG、JPEG、PNG、GIF、WebP、BMP
4. 上传后可设置：
   - **描述**: 添加图片描述
   - **公开状态**: 设为公开或私有
5. 系统会自动：
   - 生成缩略图
   - 解析 EXIF 信息
   - 根据 EXIF 生成自动标签

### 3.4 我的图片

点击导航栏上「我的图片」查看已上传的所有图片。

#### 3.4.1 筛选与搜索

- **关键词搜索**: 搜索标题、描述中的关键词
- **标签筛选**: 按标签过滤图片
- **时间筛选**: 选择上传时间范围
- **公开状态**: 筛选公开或私有图片

#### 3.4.2 批量操作

1. 点击图片的复选框选择图片
2. 选中图片后，底部会出现批量操作栏：
   - **设为公开**: 将选中图片设为公开
   - **设为私有**: 将选中图片设为私有
   - **批量删除**: 删除选中的图片

#### 3.4.3 排序

支持按以下方式排序：
- 上传时间
- 图片尺寸
- 文件大小
- 文件名

### 3.5 图片详情

点击任意图片进入详情页面。

#### 3.5.1 基本信息

- 图片预览
- 标题和描述
- 上传时间
- 文件大小和分辨率

#### 3.5.2 EXIF 信息

如果图片包含 EXIF 数据，会显示：
- 相机型号
- 拍摄时间
- 曝光参数（光圈、快门、ISO）
- 焦距
- GPS 位置

#### 3.5.3 标签管理

- 查看已有标签
- 添加新标签
- 删除标签

#### 3.5.4 AI 生成描述

1. 首先需要在「个人资料」→「AI 设置」中配置 API Key
2. 点击描述区域的「AI 生成」按钮
3. 等待 AI 分析图片内容
4. 生成的描述会自动填入描述框
5. 可手动编辑后保存

#### 3.5.5 图片编辑

点击「编辑图片」按钮进入编辑模式：

- **裁剪**: 选择裁剪区域，支持自由比例和固定比例
- **旋转**: 顺时针或逆时针旋转图片
- **翻转**: 水平或垂直翻转
- **亮度/对比度**: 调整图片明暗和对比度
- **饱和度**: 调整色彩饱和度

编辑完成后点击「保存」应用更改。

### 3.6 个人资料

点击右上角头像进入个人资料页面。

#### 3.6.1 基本信息

- **头像**: 点击头像区域将上传的图片设置为头像
- **用户名**: 修改显示名称
- **邮箱**: 查看绑定邮箱
- **简介**: 填写个人简介

#### 3.6.2 AI 设置

配置 AI 图片描述生成功能：

1. 获取 SiliconFlow API Key：
   - 访问 https://siliconflow.cn
   - 注册账号并创建 API Key
2. 在「API Key」输入框中粘贴 Key
3. 点击「保存」按钮
4. 保存后会显示脱敏的 Key（如 `sk-a...xyz`）
5. 如需删除，点击「删除」按钮

#### 3.6.3 修改密码

1. 输入当前密码
2. 输入新密码（6 位以上）
3. 确认新密码
4. 点击「修改密码」

### 3.7 管理后台（仅管理员）

管理员登录后可访问管理后台。

#### 3.7.1 图片管理

- 查看所有用户上传的图片
- 搜索和筛选图片
- 删除违规图片
- 修改图片公开状态

#### 3.7.2 用户管理

- 查看所有注册用户
- 搜索用户
- 禁用/启用用户账号
- 设置管理员权限

#### 3.7.3 系统统计

- 总用户数
- 总图片数
- 用户上传图片排行

## 4. 移动端使用

系统支持响应式布局，在移动设备上可获得良好体验：

- 底部导航栏方便操作
- 图片自适应屏幕宽度
- 支持触摸手势缩放查看图片
- 上传支持从相册选择或拍照

## 5. 常见问题

### Q1: 上传图片失败怎么办？

- 检查文件格式是否支持（JPG/PNG/GIF/WebP/BMP）
- 检查文件大小是否超过限制（默认 10MB）
- 确保网络连接正常

### Q2: 图片上传等功能失效？

请确保 Celery Worker 已启动，稍后刷新页面。


### Q3: AI 生成描述失败？

- 确认已在个人资料中配置有效的 API Key
- 检查 API Key 是否有余额
- 图片过大可能导致超时，建议使用小于 5MB 的图片

### Q4: 忘记密码怎么办？

点击登录页面的「忘记密码」链接，输入注册邮箱，系统会发送重置密码链接。

### Q5: 如何批量删除图片？

1. 进入「我的图片」页面
2. 切换到网格视图
3. 点击图片右上角的复选框选择要删除的图片
4. 点击底部的「批量删除」按钮
